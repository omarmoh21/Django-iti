## Day 2 Lab

The **Day-2 project** is a simple Django application that demonstrates relationships between different models.

### Features

* List all **Instructors**
* Click an instructor to view the **Courses they teach**
* Click a course to view the **Students enrolled**
* Use of **Django templates**
* Use of **static files (CSS & images)**
* Simple **UI styling**

### Application Flow

```
Instructors
    ↓
Courses by Instructor
    ↓
Students in Course
```
